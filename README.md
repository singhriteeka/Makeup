## 🚀 Installation

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No additional dependencies required for static HTML/CSS

### Steps
1. **Clone the repository**
   ```bash
   git clone https://github.com/singhriteeka/Makeup.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd Makeup
   ```

3. **Open the website**
   - Simply double-click `index.html`, or
   - Open it in your browser: `file:///path/to/Makeup/index.html`
   - (Recommended) Use a local server for better performance

### Using a Local Server (Optional)
```bash
# Using Python 3
python -m http.server 8000

# Using Python 2
python -m SimpleHTTPServer 8000

# Using Node.js (http-server)
npx http-server
```

Then visit `http://localhost:8000` in your browser.
